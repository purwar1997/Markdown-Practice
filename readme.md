## **To insert code snippets**

To insert keywords, use ``

In Javascript, `for` and `const` are keywords.

```javascript
for (const i in [12, 67, 90]) {
  console.log(i + 1);
}
```

> **Output** : 13 68 100

Here `let` is a keyword and `a` and `b` are variables.

---

```javascript
let sum = (a, b) => console.log(a + b);
sum(12, 1);
```

> **Output** : 13
